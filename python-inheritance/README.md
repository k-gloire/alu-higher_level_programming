# Python - Inheritance

This project is part of the ALU Higher Level Programming curriculum. It
covers inheritance in Python: superclasses and subclasses, multiple
inheritance, attribute/method lookup, and the built-in functions
`isinstance`, `issubclass`, `type`, and `super`.

## Files

| File | Description |
| --- | --- |
| `0-lookup.py` | Returns the list of available attributes and methods of an object |
| `1-my_list.py` | `MyList` class that inherits from `list`, adds `print_sorted` |
| `2-is_same_class.py` | Checks if an object is exactly an instance of a class |
| `3-is_kind_of_class.py` | Checks if an object is an instance of a class or subclass |
| `4-inherits_from.py` | Checks if an object's class inherited from a specified class |
| `5-base_geometry.py` | Empty `BaseGeometry` class |
| `6-base_geometry.py` | `BaseGeometry` with an `area` method that raises an exception |
| `7-base_geometry.py` | `BaseGeometry` with `integer_validator` |
| `8-rectangle.py` | `Rectangle` class with private width/height |
| `9-rectangle.py` | `Rectangle` class with `area` and `__str__` |
| `10-square.py` | `Square` class based on `Rectangle` |
| `11-square.py` | `Square` class with its own `__str__` |
